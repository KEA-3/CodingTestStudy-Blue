## 스터디 진행 방식

- 일주일 기준 6문제 (5문제 이상 풀기)
- 주 단위로 알고리즘 다양하게 선정
    - 정기 모임 때 **원하는 알고리즘 신청** 가능
    - 문제 담당자는 **다음 담당자를 위해 알고리즘 정리해서 공유**해주기!
- 알고리즘 분류

    ```
    그리디, 구현, 정렬
    
    이진탐색, 최단경로, 그래프이론
    
    다이나믹프로그래밍, DFS/BFS
    ```
    
- 문제 난이도
    - 백준 : 실버 5 ~ 실버 1
    - 프로그래머스 : Lv.1 (정답률 50% 이하) ~ Lv.2
- 문제 플랫폼
    - 백준 - 그룹
    - 프로그래머스 - 깃허브에 소스코드 커밋
